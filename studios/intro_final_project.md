At this point, we have covered almost all of the coding skills you will need to implement your final project. Your final project is non-trivial and something you should be starting NOW. 

In that vein, we are going to focus on getting started in class today.

For this class, there is no prior PMP due, but instead, we will review the final project together and form final project teams and have you start working on a PMP in-class.

## Review the Final Project requirements in Final Week
The directions for the Final Project are defined in the [Final Week Lab](https://mi250.info/labs/final.html). Take some time to review these instructions carefully on your own. We will also go over them in class together.

## Find a group (2-3 total) and create a PMP
Take some time to create an initial PMP that includes main objectives and take some time to think about an overall structure. You can look to your choose your own adventure studio from Week 3, but note that I expect this final project to be much more complex and interesting.
